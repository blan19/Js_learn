## BFS/DFS 3가지 패턴
### 1. 상,하,좌,우 패턴
```
dx = [1, 0, -1, 0]
dy = [0, 1, 0, -1]
```
### 2. 대각선 + 상,하,좌,우 패턴
```
dx = [1, 0, -1, 0, -1, -1, 1, 1]
dy = [0, 1, 0, -1, -1, 1, -1, 1]
```
### 3. 대각선 패턴
```
dx = [-1, -1, 1, 1]
dy = [-1, 1, -1, 1]
```
