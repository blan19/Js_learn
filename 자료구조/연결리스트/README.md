# 연결리스트

- 메모리가 허용하는한 요소를 제한없이 추가할 수 있습니다.
- 검색에는 O(N), 추가와 제거에는 O(1)이 소요된다

## Singly Linked List

- Head에서 Tail까지 단방향으로 이어지는 연결 리스트
- 가장 단순한 형태인 연결 리스트

## Doubly Linked List

- 양방향으로 이어지는 연결 리스트

## Circular Linked List

- Singly 혹은 Doubly Linked List에서 Tail이 Head로 연결되는 연결 리스트
- 메모리를 아껴쓸 수 있다.
- 원형 큐 등을 만들때도 사용된다.
